📊 Project Title
Retail Sales Analytics Dashboard (SQL + Power BI)
🔍 Problem
Businesses often struggle to extract actionable insights from large volumes of transactional data. In this project, the goal was to analyze sales data to identify revenue trends, peak sales periods, and key performance drivers to support data-driven decision-making.
📂 Dataset
Source: simulated transactional data
Size: 500K+ records
Fields include:
Order ID
Date / Time
Product Category
Revenue
Quantity
⚙️ Approach
Cleaned and transformed raw data using SQL
Performed exploratory data analysis to identify trends and patterns
Created aggregated tables for KPI tracking
Built an interactive dashboard in Power BI with filters and drill-down capabilities
🛠️ Tech Stack
SQL (data cleaning, joins, aggregations)
Power BI (dashboarding, data visualization)
Excel (initial exploration)
📈 Key Insights
Identified peak sales hours and high-performing product categories
Discovered seasonal trends affecting revenue patterns
Highlighted underperforming segments for potential optimization
🚀 Impact
Improved reporting efficiency by 20%+
Enabled faster and more accurate decision-making through interactive dashboards
Reduced reliance on manual reporting processes
