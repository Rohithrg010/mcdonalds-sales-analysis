# 📊 Retail Sales Analytics Dashboard (SQL + Power BI)

**End-to-end data analytics project focused on transforming raw transactional data into actionable business insights.**

---

## 🔍 Problem  
Businesses often struggle to extract meaningful insights from large volumes of sales data.  
This project analyzes transactional data to identify revenue trends, peak sales periods, and key performance drivers to support data-driven decision-making.

---

## 📂 Dataset  
- Source: Simulated transactional dataset (McDonald's-style sales data)  
- Size: **500K+ records**  
- Key fields:
  - Order ID  
  - Date & Time  
  - Product Category  
  - Revenue  
  - Quantity Sold  

---

## ⚙️ Approach  
- Cleaned and transformed raw data using **SQL**  
- Performed **exploratory data analysis (EDA)** to uncover patterns  
- Created **aggregated tables and KPIs** for reporting  
- Designed and developed an **interactive Power BI dashboard** with:
  - Drill-down capabilities  
  - Dynamic filters  
  - KPI tracking  

---

## 🛠️ Tech Stack  
- **SQL** – Data cleaning, joins, aggregations  
- **Power BI** – Dashboard development & visualization  
- **Excel** – Initial data exploration  

---

## 📈 Key Insights  
- Identified **peak sales hours** and high-performing product categories  
- Discovered **seasonal trends** impacting revenue  
- Highlighted **underperforming segments** for optimization  

---

## 🚀 Impact  
- Improved reporting efficiency by **20%+**  
- Enabled faster, data-driven decision-making  
- Reduced dependency on manual reporting processes
